# 1. プロジェクト概要

[Slay the Spire](https://store.steampowered.com/app/646570/Slay_the_Spire) というゲームのプレイ履歴(json)を解析して統計的なデータを表示するためのツール。

Electron 上に TypeScript + React による SPA を乗せている。

自分のためだけに作ったツールだが、当初欲しい機能は全部作ったので記念用に公開。
